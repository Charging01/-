## DailyFresh
**天天生鲜**：小型电商购物网站

项目设计使用多为django提供的API，后台管理为django-admin。适合接触django不久之后，作为第一个实战小项目。

### Introduction
- 商品浏览
- 全文检索
- 登录注册
- 用户中心
- 商品下单
- 后台管理
